# Simple Habit Tracker

A lightweight, client‑side habit tracking web app that stores your daily progress locally. Mark habits as done for the last 7 days, view your completion percentage, and keep your data after refreshing the page.

## Features

- ✅ Add new habits with a simple input field
- 📅 Each habit has checkboxes for the past 7 days (today + previous 6 days)
- 💾 Automatic local persistence – data survives page reloads
- 📊 Progress bar & percentage per habit (completed days / 7 days)
- 🗑️ Delete individual habits
- 📱 Fully responsive – works on desktop and mobile
- 🎨 Clean UI with soft colors, rounded corners, and good spacing

## Technologies Used

- **TypeScript** – for type‑safe logic and maintainable code  
- **CSS3** – custom styling, no external frameworks  
- **HTML5** – semantic markup  
- **JavaScript** (compiled from TypeScript) – runs entirely in the browser  
- **localStorage** – persists habit data locally

## How to Run the App

- npm install
- npm run dev
